# Certifications & Training

This repository contains certifications and completed training programs related to cybersecurity, penetration testing, and security operations.

All certifications were earned through legitimate training platforms and practical lab exercises.

---

## Platforms

### TryHackMe
Hands-on cybersecurity training covering:
- Networking fundamentals
- Linux & Windows security
- Web exploitation
- Privilege escalation
- SOC fundamentals

### Hack The Box
Advanced penetration testing labs focusing on:
- Realistic attack simulations
- Enumeration & exploitation
- Web application vulnerabilities
- Post-exploitation techniques

### Hacker101
Web security training covering:
- Web vulnerabilities
- Secure coding concepts
- Practical exploitation labs


---

## Continuous Learning
I actively continue improving my skills through structured labs, CTFs, and real-world simulations to strengthen both offensive and defensive security capabilities.
